编译

然后默认的go install就是在该GOPATH中你的src/xxx

你只需要go install xxx就可以了。

运行

go install生成的可执行文件默认就在GOBIN中。

然后运行时只需要命令$GOBIN/xxxx就可以了。

自定义库

和install一样的默认位置，go build xxx即可。

不生成任何文件，只有在go install某个程序时，且引用到了这个库，就会在GOPATH/pkg/中有对应的xxx.a文件。
